# 一、产品背景与目标

## 1\.1 产品背景

随着社会节奏加快，心理健康问题日益受到关注。据《中国国民心理健康发展报告》显示，超过三成的成年人存在不同程度的情绪困扰，而情绪自我觉察与管理能力的缺失是主要诱因之一。

当前市场上的情绪管理类应用普遍存在以下痛点：

- 记录流程繁琐，用户难以坚持每日打卡

- 数据呈现单一，缺乏多维度的情绪趋势分析

- 建议泛化，无法结合用户具体情况给出个性化指导

- 缺少社交与激励机制，用户留存率低

基于此，我们计划开发一款「情绪可视化管理系统」，通过轻量化的记录方式、直观的数据可视化以及AI驱动的智能建议，帮助用户建立情绪自我觉察习惯，提升心理健康水平。

## 1\.2 产品目标

### 短期目标（上线后3个月）

- 注册用户数达到 10,00\+

- 用户日均情绪记录率 ≥ 60%

- 7日留存率 ≥ 35%

- 用户满意度评分 ≥ 4\.2/5\.0

### 长期目标（上线后12个月）

- 累计用户数达到 10,000\+

- 成为国内情绪管理领域 Top 20 应用

- 建立完善的情绪数据模型，AI建议准确率 ≥ 80%

- 形成"记录\-分析\-改善"的完整情绪管理闭环

## 1\.3 产品定位

一款面向大众用户的轻量化情绪管理工具，以"可视化"为核心特色，通过数据图表直观呈现情绪变化规律，结合AI智能分析提供个性化改善建议，帮助用户科学管理情绪、提升心理韧性。

# 二、目标用户

## 2\.1 用户画像

|维度|特征描述|用户需求|
|---|---|---|
|**年龄**|18\-35岁为主，核心人群20\-28岁|追求自我提升，关注心理健康|
|**职业**|大学生、职场新人、白领|面临学业/工作压力，需要情绪出口|
|**性别**|女性占比约65%，男性约35%|女性更倾向情绪表达与记录|
|**地域**|一二线城市为主|生活节奏快，压力源多|
|**心理特征**|有自我觉察意识，愿意尝试新事物|希望了解自己、改善情绪状态|

## 2\.2 核心用户群

### 核心用户A：焦虑的职场新人

- **典型特征：**22\-26岁，刚入职场1\-2年，工作压力大，经常感到焦虑迷茫

- **核心需求：**记录每日情绪变化，找到情绪波动规律，获得调节建议

- **使用场景：**每晚睡前花3分钟记录当天心情，周末查看一周情绪总结

### 核心用户B：自我提升的大学生

- **典型特征：**18\-22岁，关注个人成长，有写日记/手账习惯

- **核心需求：**可视化的情绪记录，有趣的打卡激励，社交分享功能

- **使用场景：**课间或午休时快速记录心情，喜欢分享情绪周报到朋友圈

### 核心用户C：情绪敏感人群

- **典型特征：**25\-35岁，情绪波动较大，有轻度抑郁或焦虑倾向

- **核心需求：**长期追踪情绪趋势，专业的改善建议，预警机制

- **使用场景：**每天多次记录情绪变化，定期查看情绪趋势分析

## 3\.3 核心模块详细拆解

### 3\.3\.1 情绪记录模块

作为系统的核心输入模块，负责采集用户情绪数据，设计原则是"轻量、快速、可扩展"，确保用户能在30秒内完成一次完整记录。

|子功能|优先级|功能说明与交互细节|
|---|---|---|
|**情绪类型选择**|P0|提供8种基础情绪：开心、平静、焦虑、低落、愤怒、疲惫、惊喜、烦躁；采用大图标\+文字的卡片式布局，点击即选中；支持长按查看情绪释义|
|**情绪强度评估**|P0|5档强度滑块（1\-5分），默认居中3分；滑块颜色随强度变化（弱→强：浅蓝→深蓝/浅红→深红）；实时显示当前强度文字描述（轻微/一般/较强/强烈/极致）|
|**文字补充记录**|P0|富文本输入框，支持最多500字；提供"今天发生了什么？""是什么让你有这种感觉？"等引导性提示语；支持保存常用短语模板|
|**图片上传**|P1|支持拍照或从相册选择，最多3张；图片自动压缩保存；可添加图片说明文字；支持图片模糊处理保护隐私|
|**语音记录**|P2|支持最长60秒语音输入；自动转文字（可选）；语音文件加密存储|
|**快捷标签**|P1|预设20\+场景标签（工作、学习、恋爱、家庭、健康、社交等）；用户可多选，快速标记情绪诱因；支持自定义新增标签|
|**记录时间设置**|P1|默认记录当前时间；支持修改记录时间（补记功能）；同一天可多次记录，按时间线展示|

以日历为载体直观呈现情绪记录情况，帮助用户快速回顾一段时间的情绪状态，建立"看见情绪"的觉察习惯。

|子功能|优先级|功能说明与交互细节|
|---|---|---|
|**月视图日历**|P0|标准月历布局，每天一个格子；已记录的日期用色块填充，颜色对应情绪类型（开心=黄色、平静=绿色、焦虑=橙色、低落=蓝色等）；未记录的日期显示空白灰色|
|**爱心标记**|P1|每天显示1\-5颗爱心，爱心数量对应情绪强度；爱心颜色对应情绪类型；支持切换"色块模式"和"爱心模式"两种展示风格|

|**连续打卡标识**|P1|连续记录的日期用连线或特殊边框标记；显示当前连续打卡天数；达成里程碑（7天/30天/100天）时显示勋章动效|
|**月份切换**|P0|左右滑动切换月份；顶部显示当前年月；支持快速跳转到指定月份；显示当月记录天数和情绪平均分|
|**今日快捷入口**|P0|日历右下角悬浮"\+"按钮，点击快速进入记录页面；今天的日期有特殊高亮标记|

### 3\.3\.3 数据分析模块

多维度可视化呈现情绪数据，帮助用户发现情绪规律和触发模式，是产品"可视化"核心特色的主要载体。

|子功能|优先级|功能说明与交互细节|
|---|---|---|
|**情绪趋势折线图**|P0|横轴为时间（日/周/月/年可切换），纵轴为情绪分值（1\-5分）；折线颜色随情绪值变化（高=暖色，低=冷色）；支持双指缩放查看细节；悬停显示当天具体数值和情绪类型|
|**情绪分布饼图**|P0|展示各情绪类型的占比分布；点击某一切片可查看该情绪的详细统计（出现次数、平均强度、高发时间段）；支持按周/月/季度/年切换统计周期|
|**记录频率柱状图**|P0|按星期/时间段统计记录次数柱状图；帮助用户发现自己的记录习惯；显示总记录次数、日均记录数、最长连续记录等关键指标|
|**时段分析图**|P1|按一天24小时统计情绪分布；用热力图形式展示哪些时间段情绪偏高/偏低；帮助识别情绪波动的时间规律|
|**标签关联分析**|P1|分析不同标签（工作、恋爱、健康等）对应的情绪均值；用横向条形图对比展示；帮助用户识别哪些场景更容易影响情绪|
|**周期对比**|P2|支持两周对比、两月对比；双折线图叠加展示，直观看到变化趋势；自动计算变化幅度和改善率|
|**数据筛选**|P1|支持按情绪类型、标签、时间范围筛选数据；筛选后图表实时更新；支持保存常用筛选条件为快捷视图|

### 3\.3\.4 智能洞察模块

基于AI算法分析用户情绪数据，自动识别异常模式并生成个性化建议，实现从"记录"到"改善"的价值闭环。

|子功能|优先级|功能说明与交互细节|
|---|---|---|
|**异常情绪识别**|P0|自动检测连续3天以上的低落/焦虑情绪；检测单日极端情绪（强度≥4且持续多次）；检测情绪波动幅度过大的情况；识别后推送提醒卡片|
|**情绪规律洞察**|P0|自动分析并生成洞察结论，如："你每周一情绪最低，周五最高""下午3点是你一天中情绪低谷期""工作标签出现时，焦虑情绪概率增加60%"；每条洞察配数据依据|
|**个性化建议生成**|P0|基于当前情绪和历史模式，推荐匹配的调节方法；建议分类：呼吸训练、冥想引导、音乐推荐、运动建议、认知调整、社交支持等；每条建议标注预计耗时和适用场景|
|**每周情绪报告**|P1|每周一生成上周情绪报告；包含：本周情绪概览、关键发现、与上周对比、下周改善建议；支持生成分享海报；报告以卡片式排版呈现，图文并茂|
|**情绪预警机制**|P1|连续负面情绪达到阈值时，主动推送关怀提醒；严重情况推荐专业心理援助资源；预警等级分三级：关注、警惕、高危，对应不同干预策略|
|**建议效果反馈**|P1|用户执行建议后可反馈"有帮助/一般/没帮助"；系统根据反馈优化推荐算法；显示每条建议的用户好评率|
|**AI情绪对话**|P2|接入大语言模型，支持用户文字倾诉；AI进行共情回应和情绪疏导；对话记录自动关联当天情绪数据；可选择是否将对话用于模型训练|

# 四、用户使用流程

## 4\.1 核心使用流程总览

用户使用系统的完整路径分为**入门流程**和**核心使用闭环**两部分：

**一、入门流程（线性路径）**

注册登录 → 新手引导 → 个性化设置 → 首次情绪记录 → 进入首页

新用户首次使用时，依次完成账号注册、产品引导、偏好设置等步骤后，通过一次引导式的情绪记录体验核心功能，最终进入系统首页开始正式使用。

**二、核心使用闭环（循环路径）**

日常情绪记录 → 查看情绪趋势 → 获取智能建议 → 执行调节练习 → 日常情绪记录（循环往复）

用户进入系统后，在日常使用中形成完整的情绪管理闭环：先记录当下情绪状态，再通过数据可视化查看情绪变化趋势，系统基于数据分析生成个性化改善建议，用户执行相应的调节练习后，情绪状态发生变化，再次进入记录环节，形成"记录—分析—改善—再记录"的正向循环。

用户从注册到深度使用的完整路径如下：

## 4\.2 新用户首次使用流程

1. **注册登录**：支持手机号、微信、Apple ID 三种登录方式

2. **新手引导**：3页滑动引导页，介绍产品核心价值与功能

3. **个性化设置**：引导用户设置情绪目标、提醒时间、偏好标签

4. **首次情绪记录**：引导完成第一次心情打卡，体验核心功能

5. **进入首页**：展示今日情绪卡片 \+ 快捷记录入口

## 4\.3 日常情绪记录流程

1. **触发记录**：用户主动打开APP / 收到定时提醒

2. **选择情绪等级**：5档情绪滑块/表情选择，1秒完成

3. **添加情绪标签**：可选，多选标签细化情绪类型

4. **记录触发事件**：可选，简单描述或选择分类

5. **写心情日记**：可选，文字/图片/语音记录详细感受

6. **完成记录**：提交后获得即时反馈与建议

## 4\.4 查看情绪分析流程

1. **进入统计页面**：底部Tab切换到"趋势"页面

2. **选择时间维度**：切换日/周/月/年视图

3. **查看趋势图表**：折线图展示情绪变化曲线

4. **多维度筛选**：按标签、事件类型筛选查看

5. **查看分析解读**：AI生成的情绪分析与改善建议

6. **生成/分享报告**：可生成周报/月报，支持分享到社交平台

## 4\.5 获取智能建议流程

1. **记录后即时建议**：每次情绪记录后，根据当前情绪推送调节建议

2. **主动浏览建议库**：在"建议"页面按分类浏览各类情绪调节方法

3. **执行建议练习**：呼吸训练、冥想引导、音乐疗愈等

4. **反馈效果**：练习后可反馈是否有效，优化AI推荐模型
